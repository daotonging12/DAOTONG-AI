## 一句话定位

DAOTONG-AI 是一套**开源、可审计、宪法约束的 AI 治理协议栈**——让 AI 在明确的规则边界内辅助公共决策，降低腐败与低效，同时保留人类的最终否决权。

## 三层架构

```
┌─────────────────────────────────────────────────┐
│          全民共识层 (CCL)                         │
│  链上投票 · 公众监督 · 宪法修订权                  │
├─────────────────────────────────────────────────┤
│          元老院 (SOW)                             │
│  随机抽选公民代表 · 否决权 · 紧急停止权             │
├─────────────────────────────────────────────────┤
│          人工智能治理核心 (AIGC)                   │
│  规则执行 · 行政裁决 · 审计追踪                    │
│  宪法硬约束 · 模型可替换 · 全链路审计               │
└─────────────────────────────────────────────────┘
```

详细架构说明见 [docs/architecture/ARCHITECTURE.md](docs/architecture/ARCHITECTURE.md)。

## 核心原则

- **不发币、不 ICO、不出售 NFT 公民身份** — 治理不金融化
- **开源可审计** — 核心代码永久开源，任何人可审计，任何人可 fork
- **人类终裁** — AI 辅助决策，人类保留否决权
- **诚实透明** — 所有公开内容反映真实开发进度
